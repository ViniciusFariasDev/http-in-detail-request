# HTTP in Detail

This repository documents my hands-on learning related to **HTTP fundamentals**, focusing on how web communication works between clients and servers.

## Topics Covered

- HTTP vs HTTPS
- Request and response structure
- HTTP methods (GET, POST, PUT, DELETE)
- HTTP status codes (2xx, 4xx, 5xx)
- Headers and cookies
- Sending parameters in request bodies
- Practical interaction with web endpoints
